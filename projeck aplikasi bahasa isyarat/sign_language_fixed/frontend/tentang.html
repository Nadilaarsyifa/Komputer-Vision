Latar Belakang

Tentang Proyek Ini

SignLens lahir dari keprihatinan akan kurangnya aksesibilitas digital bagi komunitas Tuli di Indonesia. Bahasa Isyarat Indonesia (BISINDO) adalah bahasa utama komunitas Tuli, namun masih sangat sedikit sistem teknologi yang mendukungnya.

Proyek ini mengembangkan sistem pengenalan BISINDO berbasis kamera web yang dapat berjalan langsung di browser tanpa perlu instalasi aplikasi. Dengan memanfaatkan computer vision dan machine learning, tangan pengguna dianalisis secara real-time untuk diidentifikasi sebagai huruf, angka, atau kata.

Tiga mode deteksi tersedia: huruf A–Z menggunakan kedua tangan, angka 0–9, dan 7 kata menggunakan analisis tangan + wajah + pose tubuh secara holistik.

  • Deteksi berbasis landmark bukan gambar mentah — lebih ringan dan privasi-friendly
  • Tiga mode deteksi: huruf, angka, dan kata dengan model ML terpisah
  • Smoothing prediksi menggunakan sliding window (deque) untuk hasil yang stabil
  • Threshold confidence 50% — prediksi dengan keyakinan rendah diabaikan
  • Fallback simulasi otomatis jika kamera atau backend tidak tersedia
  • Panduan gesture visual tersedia langsung di dalam aplikasi
  • Teks terbentuk otomatis dari kumpulan huruf/kata yang terdeteksi
  • Backend FastAPI mudah di-extend untuk mendukung lebih banyak kelas
