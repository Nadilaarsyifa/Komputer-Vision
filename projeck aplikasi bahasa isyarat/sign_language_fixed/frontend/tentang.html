Latar Belakang

Tentang Proyek Ini

SignLens merupakan sistem pengenalan gesture bahasa isyarat Indonesia berbasis computer vision dan machine learning yang dikembangkan sebagai project pembelajaran mata kuliah Computer Vision dan Machine Learning.

Sistem memanfaatkan kamera web untuk mendeteksi gesture secara real-time melalui browser.

Project ini dirancang sebagai implementasi teknologi AI berbasis gesture sekaligus eksplorasi penerapan computer vision dalam interaksi manusia dan komputer.

  • Menggunakan pendekatan landmark sehingga proses deteksi lebih ringan dibanding pengolahan citra mentah
  • Menggabungkan teknologi computer vision dan machine learning dalam satu sistem terintegrasi
  • Antarmuka dirancang sederhana, responsif, dan mudah digunakan
  • Prediksi gesture ditampilkan secara langsung melalui kamera web
  • Sistem dirancang agar mudah dikembangkan untuk penambahan gesture baru
